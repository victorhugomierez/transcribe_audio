## 🎯 Overview
**Omni-Analyzer** is an advanced natural language processing (NLP) tool designed for extracting tactical and academic intelligence. The tool allows you to transform audio, PDF, and Word documents into strategic reports with topic intensity analysis.

## 🚀 Key Features (v3.6)
- **Summary Module:** Sentence punctuation algorithm to generate executive summaries.

- **Active Recall Engine:** Automatic generation of self-assessment questions (Q&A) based on term frequency.

- **Concept Mapping:** Identification of relationships between critical categories (e.g., Pedagogy vs. Institutions).

- **Multi-Source:** Support for Whisper (Audio/Video), PyMuPDF (PDF), and Docx.

- **Universal Mode:** RAW analysis for keyword discovery in unstructured environments.
